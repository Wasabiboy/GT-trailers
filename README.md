# GT Trailers Website

A modern, responsive website for GT Trailers built with HTML, CSS, and JavaScript, based on the Indofact industrial theme.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Hero Slider** - Auto-playing image slider with manual controls
- **Smooth Scrolling** - Navigation with smooth scroll effects
- **Project Filter** - Filter projects by category
- **Animated Counters** - Statistics counters with scroll-triggered animations
- **Contact Form** - Functional contact form (ready for backend integration)
- **Mobile Menu** - Hamburger menu for mobile devices

## Sections

1. **Hero Section** - Full-screen slider with call-to-action
2. **Features** - Three key feature cards (Expert Craftsmanship, Quality Materials, Reliable Trailers)
3. **About** - Company information with statistics
4. **Services** - Six trailer service offerings (Boat Trailers, Utility Trailers, Custom Trailers, etc.)
5. **Stats** - Animated statistics counter
6. **Products** - Product showcase (Boat Trailers, Utility Trailers, Trailer Hardware)
7. **Contact** - Contact form and information
8. **Footer** - Site links and information

## How to Use

1. Simply open `index.html` in a web browser
2. No build process or dependencies required
3. All images are included in the `images/` folder
4. Uses Font Awesome CDN for icons (internet connection required)

## Customization

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #e63946;
    --secondary-color: #1d3557;
    --dark-color: #1a1a1a;
    /* ... */
}
```

### Content
- Edit `index.html` to change text content
- Replace images in the `images/` folder
- Update image paths in HTML if needed

### Functionality
- Modify `js/main.js` to add or change interactive features
- Contact form currently shows an alert - integrate with your backend
